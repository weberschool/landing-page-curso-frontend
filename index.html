Resumo

Entenda em 4 semanas os princípios de como funciona a Web e adquira conhecimentos práticos nas três tecnologias que fundamentam todos os websites: HTML, CSS e JavaScript.

Duração

O curso Desenvolvedor Web Front-end foi dividido em 04 módulos semanais que irão do básico ao intermediário, mas permeia por alguns recursos avançados da Web. O curso é 100% prático, baseado em projetos, e segue uma metodologia moderna de ensino que permite transformar você em um desenvolvedor Front-End desejável pelo mercado de trabalho. Conheça os detalhes de cada dia

1

A Web do jeito certo!

O foco será entender a Web, sua história, o estado atual e a importância do desenvolvedor front-end para grandes empresas. Além disso, exploraremos as principais tecnologias da Web, como por exemplo: HTML5, CSS3, Web Semântica, Acessibilidade e Metatags. O objetivo dessa semana é montar um website com suas informações para contato e publicar no github pages.

  • Data: 20/01 e 21/01 (9h às 18h)
2

Desvendando o CSS3

Nessa etapa serão explorados recursos avançados do CSS3, que permitirão você montar sites inscríveis a fim de melhorar a experiência do usuário. A meta para esse final de semana será deixar sua página HTML idêntica ao design que disponibilizaremos durante o curso, ele possui diversos detalhes que serão todos reproduzidos usando apenas CSS3.

  • Data: 10/02 e 11/02 (9h às 18h)
3

Integração com APIs e Ajax

Adicionaremos recursos avançados à nossa página Web usando a linguagem de programação JavaScript. Mostraremos como utilizar os recursos de Geolocation e HttpRequest pra tornar sua página rica e dinâmica. O projeto dessa etapa será uma busca com autocomplete utilizando as APIs do Google Maps a partir de um endereço e adicionar marcadores a ele.

  • Data: 24/02 e 25/02 (9h às 18h)
4

Construindo uma WeberFlix

Etapa onde serão consolidados todo o conhecimento adquirido, além de aplicar algumas técnicas de alta performance. Desenvolveremos um catálogo do Netflix com alguns recursos avançados. Você será capaz de desenvolver o projeto com qualidade e velocidade, requisitos essenciais para quem deseja trabalhar em grandes empresas como Desenvolvedor Web Front-end.

  • Data: 03/03 e 04/03 (9h às 18h)

AHistória da Web

Você conhecerá a história por trás das linguagens que serão ensinadas neste curso, a Web e da nossa posição atual como desenvolvedores Web.

BHTML e CSS

Você aprenderá como funciona a linguagem de marcação HTML e seus principais recursos. Saberá como estilizar seus documentos HTML usando a linguagem CSS para que fiquem bonitas e agradáveis.

CWeb Semântica e Acessibilidade

Você aprenderá como organizar a informação das suas páginas de maneira legível para computadores e máquinas poderem trabalhar em cooperação com os humanos. Além de torna-los acessíveis para que todos os usuários podem ter igual acesso à informação e funcionalidade.

DPublicando seu site na internet

Ensinaremos você a disponilizar um website gratuitamente no Github Pages para que mostre aos amigos e familiares. Diga adeus ao FTP, usaremos a ferramenta Git pra colocar no servidor Web.

Desvendando o CSS3

O curso Desenvolvedor Web Front-End vai mergulhar no mundo do CSS3. Ensinará você a usar os principais recursos que deixarão suas páginas Web idênticas ao PSD fornecido pelo designer. Depois desse módulo você será capaz de:

  • Realizar efeitos de transição usando apenas CSS
  • Deixar o site igual ao que foi desenhado
  • Tornar sua página confortável no desktop, mobile e tablet
